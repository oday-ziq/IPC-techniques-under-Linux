** Message Format **
    - Drop x Containers: "drop x"
    - Kill plane: "killplane"
    - Collect Container: "collect"
    - Split Container: "split"
    - Send Wheat bags to families: "send"
    - Collecting Worker Died: "collectdie"
    - Distributing Worker Died: "distributedie"
    - Add a worker on distribute from split: "adddis"
    - Add a worker on collect from split: "addcol"


** Info on UI **
    - Count containers dropped to the ground
    - Count container in the Safe Area
    - Count Wheat Bags in the Safe Area
